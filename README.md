# Portafolio Web

## Descripción

Este proyecto consiste en un portafolio web responsivo desarrollado para presentar mi perfil como desarrolladora Full-Stack Junior. Incluye una sección de presentación personal, habilidades técnicas, proyectos destacados y un apartado de contacto.

La página se adapta a diferentes tamaños de pantalla (escritorio, tablet y móvil) mediante técnicas de diseño responsivo, ofreciendo una experiencia de navegación clara y organizada.

## Tecnologías y técnicas utilizadas

- HTML5 con estructura semántica.
- CSS3 para estilos y diseño visual.
- CSS Grid para la estructura principal de la página.
- Flexbox para la alineación y distribución de elementos.
- Media Queries para adaptar el diseño a distintos dispositivos.
- Tipografías personalizadas mediante `@font-face`.
- Imágenes optimizadas con `object-fit`.
- Organización del proyecto siguiendo buenas prácticas de desarrollo.
